# auto_java8_installer
java 8 auto install script

just run:

```shell

sudo chmod +x java_8_install_script.sh


./java_8_install_script.sh

```

then enjoy you java coding life 233

中文使用方法：

下载或者clone项目至本地

只需要在终端运行:

```shell

sudo chmod +x java_8_install_script.sh


./java_8_install_script.sh

```

然后就享受你的java
